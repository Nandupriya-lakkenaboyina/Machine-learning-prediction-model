# Student Placement Prediction using Machine Learning

## Project Overview

This project predicts student placement status using machine learning techniques. The dataset contains student academic, demographic, and skill-related information. The goal is to build a classification model that can accurately predict whether a student will be placed.

## Features

* Data preprocessing and cleaning
* Handling missing values
* Encoding categorical variables
* Feature scaling
* Train-test split
* Model training and evaluation
* Accuracy calculation
* Confusion Matrix visualization
* ROC Curve and AUC analysis

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Project Workflow

1. Load the dataset
2. Handle missing values
3. Convert categorical data to numerical format
4. Scale numerical features
5. Split data into training and testing sets
6. Train the machine learning model
7. Evaluate model performance using:

   * Accuracy Score
   * Confusion Matrix
   * ROC Curve

## Results

The model was evaluated using standard classification metrics and visualization techniques to measure prediction performance.
